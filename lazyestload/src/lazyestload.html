Scroll up and down the page. Try scrolling past some of the images

Images will only load when they are in the viewport and when the user has stopped scrolling (for longer than 0.2 seconds)

Meaning that the user can scroll past images, and they wont be loaded.

Some of these images are large ~1mb for demo purposes. You can also open up your dev tools to see when the browser requests the images, it might also help to throttle the bandwidth if the images are loaded to quickly to see the change.
